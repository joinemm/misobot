# Miso Bot
A discord bot written in Python using the rewrite branch of discord.py library. Has tons of API integrations and all sorts of fun commands to use.

# REST APIs used
- LastFM
- Spotify
- Google maps & Google translate
- Dark sky
- Twitter
- Oxford Dictionaries
- Urban Dictionary
- Naver Translate
- Ucum unit transform
- CurrencyConverter
- TheColorApi
- Wolfram Alpha
- Steam
- TimezoneDB
