Info

Miso Bot is a discord bot written in python using the rewrite branch of discord.py.

Invite Miso to your server (soon) Gimme money (patreon)

Basic commands

Info

Get information about the bot.

Usage: >info

Ping

Get the bot's ping.

Usage: >ping

Uptime

Get the bot's uptime.

Usage: >uptime

Random number generator

Gives random number from given range. defaults to 1

Usage: >random [number]

Youtube

Search for a youtube video.

Usage: >youtube [search query]

Wikipedia

Search for a page on wikipedia.

Usage: >wikipedia [search query]

Stan

Get a random kpop artist.

Usage: >stan

Navyseal

The navy seal copypasta.

Usage: >navyseal

Ascii

Turn text into fancy ascii art.

Usage: >ascii [text]

Instagram

Pulls all the source images from instagram post in the highest quality possible.
For videos use >igvideo.

Usage: >ig [url]

Youtube to mp3

Downloads the audio from a youtube video and posts it as an mp3 file

Usage: >ytmp3 [url]

API commands

Weather

Get weather of a given location. Uses Google's location search api so the search is flexible.

Usage: >weather [location]

Define

Get a word's definition from The Oxford dictionary.

Usage: >define [word]

Weather

Get a word's definition from Urban dictionary.

Usage: >urban [location]

Translate

Translates text. Supports EN->KR, KR->EN and JP->EN. Detects language automatically.

Usage: >translate [text]

Spotify

Analyzes a spotify playlist and gets the artist distribution in it. URI preferred but url works too.

Usage: >spotify [URI or url]

Convert

Converts various units (Celsius to Fahrenheit is inaccurate don't use it).

Usage: >convert [amount] [unit_1] to [unit_2]

Currency

Converts currency. Use three letter currency codes like EUR or USD.

Usage: >currency [amount] [currency_1] in [currency_2]

Steam

Get steam profile data

Usage: >steam [steam id]

Color

Get a representation of a hex color, user's color, or a random color.

Usage: >color [hex | @mention | random]

LastFM

fm

Get data from you last fm account. Optionally use chart to generate an image (takes some time).

Optional arguments

datatype: [nowplaying np | recent re | toptracks tt | topartists ta | topalbums talb]

timeframe: [week | month | 3month | halfyear | year | overall]

chart: [chart 3x3 | chart 4x4 | chart 5x5]

Usage:
>fm [datatype] [timeframe] [chart]
>fm [datatype]
>fm

fm geo

Get lastfm data from a specific country.

Arguments

datatype: [nowplaying np | recent re | toptracks tt | topartists ta | topalbums talb]

Usage: >fmgeo [datatype] [country]

fm data

Get lastfm data about artist, album, or track.

Arguments

datatype: [artist | album | track]

Usage:
>fmdata artist [artist]
>fmdata album [album] by [artist]
>fmdata track [track] by [artist]

User

Profile

Gets your profile. Optionally mention someone to get their profile

Usage: >profile [@mention]

Avatar

Gets your discord avatar. Optionally mention someone to get their avatar

Usage: >avatar [@mention]

Fishy

Go fishing. Receive random fish. Optionally mention someone to fish for them.

Usage: >fishy [@mention]

Fishy stats

See statistics about your fishing. Mention someone to see their stats. Use global to see global stats.

Usage: >fishystats [@mention | global]

Fishy leaderboard

See the fishy leaderboard

Usage: >leaderboard

Moderator

Mute

Mute user (you must make a muterole first)

Usage:
>mute [@mention]
>unmute [@mention]

Config

Configure server specific parameters.

Arguments

Setting: [welcome | muterole]

Usage:
>config welcome [channel id]
>config muterole [@mention role]

Music

Join

Join the given voice channel

Usage: >join [voice_channel]

Play

Search youtube for the given song and play it. Automatically joins the voice channel you're in.

Usage: >play [search query]

Stop

Stops playback and disconnects from voice channel.

Usage: >stop
